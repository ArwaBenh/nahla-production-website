# Nahla Production Website

A responsive, interactive website built from scratch for **Nahla Production**, showcasing their services, projects, and contact information.

## Features
- Responsive navigation and dynamic menu
- Typing effect to highlight key text
- Image & video galleries
- Contact form with live validation
- Downloadable catalogue

## Technologies Used
- HTML
- CSS
- JavaScript
